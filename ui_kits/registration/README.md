# PrepPals Registration — UI Kit

A pixel-recreated lead-capture landing for CSCA PrepPals. The hero is a single
above-the-fold composition: nav + headline/subhead + feature pills + dual CTA
on the left, registration form card on the right, full-width stats strip below.

## Components
- `Nav.jsx` — top navigation with logo lockup, links, "Book Free Trial" CTA.
- `Hero.jsx` — left-side headline, subhead, feature pills, CTA pair.
- `RegistrationCard.jsx` — right-side form card with the 7 fields.
- `Field.jsx` / `Select.jsx` — input primitives shared by the form.
- `FeaturePill.jsx` — circular orange-chip + label/sub.
- `StatsStrip.jsx` — full-width 4-up stats footer.
- `MathCanvas.jsx` — decorative floating math glyphs + dot grid.
- `Button.jsx` — primary/secondary CTA.

## Design width
1440 × 900. Canvas scales down on smaller viewports.
